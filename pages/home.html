MakeMyTrip Data Pipeline using PySpark
This project implements a data pipeline for extracting, transforming, and visualizing hotel data from the MakeMyTrip website. In this pipeline, Selenium is used to scrape hotel data from the MakeMyTrip website, Apache Spark is employed to process and transform the extracted hotel datasets, the processed data is stored in a PostgreSQL database, and lastly Plotly is used to create interactive data visualizations in Jupyter notebooks.

Food Ordering App Backend with Apache Kafka
This project presents a scalable backend system for a food ordering app, leveraging the power of Apache Kafka as a central event bus. Events, such as order placements and confirmations, are written to Kafka topics, facilitating real-time processing by different components like microservices, stream processing jobs, and Python modules.

This project presents an ETL (Extract, Transform, Load) pipeline designed to summarize podcast episodes using Apache Airflow, a robust open-source workflow orchestration platform. The workflow is tailored for SQLite3, providing a simple yet effective solution for managing podcast data.
